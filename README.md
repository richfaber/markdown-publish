
# markdown publish tool

## 소개

마크다운(md) 파일을 로컬에서 편집하고, 블로그에 포스팅 하는 도구. 


## 주요 기능

- 편집
- 미리보기
- 블로그 포스팅
- 이미지 클라우드 업로더 google photo / 


## Quick start

- `git clone https://github.com/richfaber/markdown-puslish.git` : 프로젝트 파일 다운로트
- `npm install` : 의존성파일 설치
- `npm run start` : 편집기 구동
